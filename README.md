# 🎤 SpeakAce – AI Communication Practice Platform

SpeakAce is a web-based AI-inspired communication practice platform designed to help students prepare for placement interviews such as Cognizant SVAR, TCS HR rounds, Capgemini communication rounds, and more.

It provides interactive speaking and listening practice with real-time speech recognition and performance evaluation.

---

## 🚀 Features

### 🧑‍💻 Authentication Flow
- Login page
- Signup page

### 📊 Dashboard System
- Company-based practice selection
- Easy navigation to different interview rounds

### 🎯 Practice Modules

#### 🎤 Speaking Round
- Random topic generation
- 60-second timer
- Speech recognition using Web Speech API
- Real-time speech transcription

#### 👂 Listening Round
- Text-to-speech audio playback
- Sentence repetition practice
- Speech recognition for answers
- Accuracy checking system

#### 🧠 HR Interview Practice
- Common HR questions
- Interview preparation flow

---

## 📈 Performance Report System

- Automatic scoring system based on:
  - Fluency
  - Pronunciation
  - Confidence
  - Listening ability

- Smart rule-based evaluation system
- Personalized improvement suggestions
